## Front-Yard Savanna

A 1,500-square-foot oak-savanna-inspired front yard for a Twin Cities lot. Replaces conventional turf with native grasses, sedges, and wildflowers across three moisture zones, with a Bur Oak anchor tree, Serviceberry understory, and 18 herbaceous species providing continuous bloom from April through October.

## How These Designs Were Made

Each design is generated programmatically from Python scripts that combine the site plan logic, plant schedules, and per-species data with the same images that appear in the [Plant Gallery](../plants/index.md). The source scripts live in the [project's `landscape-design/` directory on GitHub](https://github.com/arunbatchu/minnesota-native-plants/tree/main/landscape-design) — adapt them to your own lot, sun exposure, and design goals.
