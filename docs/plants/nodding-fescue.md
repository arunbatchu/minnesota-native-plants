# Nodding Fescue

*Festuca subverticillata*



_No photos available yet. See [Learn More](#learn-more) links below._

Nodding Fescue (*Festuca subverticillata*) is a graceful, cool-season native grass of shaded and partly shaded ground. Its delicate flowering stalks arch and nod, with the tiny spikelets held on fine, spreading branches that shimmer in a breeze. Because it tolerates shade, it bridges the woodland edge and the shaded meadow — and it is one of the grasses seeded into the meadow at the Eloise Butler Wildflower Garden.

## Quick Facts

| | |
|---|---|

| **Bloom time** | May–July |
| **Sun** | Part shade to full shade |
| **Moisture** | Medium |
| **Soil** | Rich woodland and woodland-edge soils |
| **Wildlife value** | Cover for small wildlife; larval host for some skipper butterflies |
