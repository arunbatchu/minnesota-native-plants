# Spring Beauty

*Claytonia virginica*



_No photos available yet. See [Learn More](#learn-more) links below._

Spring Beauty (*Claytonia virginica*) is one of the earliest woodland wildflowers to appear in a Minnesota spring. It is a low spring ephemeral — a plant that leafs out, flowers, and goes dormant before the forest canopy fills in. Its small, five-petaled white-to-pink flowers are striped with darker pink veins that guide early native bees to nectar. You can see drifts of it carpeting the woodland floor at the Eloise Butler Wildflower Garden in April and May.

## Quick Facts

| | |
|---|---|
| **Scientific name** | *Claytonia virginica* |
| **Family** | Montiaceae (purslane family) |
| **Height** | 4–6 inches |
| **Bloom time** | April–May |
| **Sun** | Part shade to full shade |
| **Moisture** | Medium |
| **Soil** | Rich, moist woodland soil |
| **Wildlife value** | Early nectar and pollen for native bees, including the spring-beauty specialist bee (*Andrena erigeniae*) |
