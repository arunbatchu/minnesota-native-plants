# Cranberry

*Vaccinium macrocarpon*



_No photos available yet. See [Learn More](#learn-more) links below._

Vaccinium macrocarpon, also called large cranberry, American cranberry and bearberry, is a North American species of cranberry in the subgenus Oxycoccus.
The name cranberry comes from the shape of the flower stamen, which looks like a crane's beak.
