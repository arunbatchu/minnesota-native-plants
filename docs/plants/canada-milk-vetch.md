# Canada Milk-Vetch

*Astragalus canadensis*



_No photos available yet. See [Learn More](#learn-more) links below._

Canada Milk-Vetch (*Astragalus canadensis*) is a tall, upright prairie wildflower in the pea family. In mid- to late summer it sends up dense spikes of creamy greenish-white, pea-like flowers, which give way to clusters of plump seed pods. Like other legumes, it works with soil bacteria to pull nitrogen from the air and enrich the soil around it. It is one of the wildflowers seeded into the meadow at the Eloise Butler Wildflower Garden.

## Quick Facts

| | |
|---|---|
| **Scientific name** | *Astragalus canadensis* |
| **Family** | Fabaceae (pea/legume family) |
| **Height** | 2–4 feet |
| **Bloom time** | July–August |
| **Sun** | Full sun to part shade |
| **Moisture** | Medium to moist |
| **Soil** | Adaptable; enriches soil by fixing nitrogen |
| **Wildlife value** | Nectar for long-tongued bees; larval host for several butterflies; seeds eaten by wildlife |
