## Site facts

- **Lot:** 0.36 acres (~15,700 sq ft) · Parcel ID 1311622340067 (Hennepin County)
- **Built:** 1997 · 5,500 sq ft, 2 stories · tan stucco / wood-shake
- **Exposure:** South-facing, full sun (>6 hr); stucco wall reflects heat → effective USDA zone 5a microclimate
- **Pre-settlement vegetation:** Big Woods / prairie-forest border subsection — bur oak savanna on rolling moraine. *This design is literally a small reconstruction of pre-1850 Hennepin County.*
- **Soil (likely):** Hayden / Lester / Kilkenny loam (Hennepin moraine glacial till). Confirm via [USDA Web Soil Survey](https://websoilsurvey.nrcs.usda.gov/app/) before install.

## Design summary

A small pocket of Minnesota savanna anchored by a Serviceberry on the west end, Ninebark shrubs flanking the bay window kept low for the indoor sightline, Gray Dogwood tucked behind the Serviceberry, and drifts of native prairie forbs in warm savanna tones connected by Little Bluestem and Prairie Dropseed grasses. A meandering limestone path enters from the east walkway and arrives at a partially-shaded bench pocket in the west third.

The existing river-rock drainage swale becomes a fully-vegetated bioswale (Rima's preference: no rock) — sedges and rushes form a fibrous root mat that armors the flow path while looking like a meadow.

**Key constraints honored:**

- Two existing arborvitae at east end → preserved, anchor that side
- Existing crabapple at front entry → harmonized (white serviceberry bloom complements pink crabapple)
- Boxwood/yew hedge at front porch → out of scope, untouched
- Bay-window indoor sightline → central zone kept under 30" mature
- HVAC and gas meter on west wall → 3 ft service clearance maintained
- Rima's no-rock preference → vegetated swale instead of dry-creek

## Re-rendering & iteration

Every visual deliverable has a Python source. To tweak any element:

```bash
cd landscape-design/front-yard-savanna

# Drawn plans (no API needed)
python3 annotate.py        # photo overlays
python3 site_plan.py       # top-down plan
python3 sections.py        # cross-sections
python3 details.py         # construction details
python3 gantt.py           # bloom calendar

# Photoreal renderings (needs GEMINI_API_KEY in .env)
python3 render.py year3            # one target
python3 render.py year1 year5      # multiple targets
python3 render.py                  # all four
```

Edit the constants near the top of any script (zone polygons, species lists, marker positions, prompts) and re-run.

The `.env` file holds the Gemini API key and is `.gitignored` — verify with `git check-ignore -v landscape-design/front-yard-savanna/.env` before any commit.
